## Installation
```bash
cd ...
%APPDATA% # on Windows
$XDG_CONFIG_HOME or ~/.config # on Linux
~/Library/Application Support # on OS X
mkdir iron-node;
npm install iron-node-zerodarkmatrix-theme;
```
