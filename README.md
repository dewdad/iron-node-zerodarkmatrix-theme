## Installation


```bash
# be shure you uninstalled other themes

# on Windows
cd %APPDATA%

# on Linux
cd $XDG_CONFIG_HOME
# or
cd ~/.config;

# on OS X
cd ~/Library/Application Support;

# create if not exists
mkdir iron-node;

# Finaly
cd iron-node;
npm install iron-node-zerodarkmatrix-theme;

# restart iron-node
```
